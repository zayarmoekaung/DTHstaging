ဒစ်ဂျစ်တယ်လုံခြုံရေး အလေ့အထကောင်းများ ကျင့်သုံးခြင်း

မိိမိဖုန်းနှင့် ကွန်ပျူတာများအား တစ်ပါးသူများအားဖြစ်နိုင်လျှင် ပေးမသုံးပါနှင့် မိိမိအကောင့်များ၏ password များကို ခိုင်မာစွာထားရှိပါရန်။ မိမိ စကားဝှက် Password များအား မည်သူ့ကိုမျှ မပေးပါနှင့် ။ Password များကို အနည်းဆုံး ၆လတစ်ခါပြောင်းပေးရန်။ မိမိ၏ကိုယ်ရေးကိုယ်တာအချက်အလက်များကို အွန်လိုင်းပေါ်တွင်အလွယ်တကူဖော်ထုတ်ပြသခြင်း မပြုမိပါရန်။ မိမိ၏ Gmail နှင့် Facebook Account များကို Two Factor Authentication ခံထားသင့်သည်။ မိမိအသုံးပြုနေသော Browser History များကို ရှင်းလင်းရန်။ အရေးကြီးသော အချက်အလက်များကိုပြောဆို‌‌‌ဝေမျှသောအခါ စိတ်ချရသော Platform များတွင် ပြောဆိုသင့်သည်။ Public Wifi များကို ချိတ်ဆက်အသုံးပြုရာတွင် မိမိအရေးကြီးသောကိုယ်ရေးကိုယ်တာ သို့ ငွေကြေးဆိုင်ရာအကောင့်များ၏ password များကိုအလွယ်တကူရိုက်ထည့်ခြင်းကိုရှောင်ကြည်ရန်။ ဖုန်းဆိုင်များတွင် ဖုန်းပြုပြင်သည့်အခါ ( သို့ ) မိမိဖုန်းအား တစ်ပါးသူသို့ ပြန်လည်ရောင်းချသည့် အခါတွင် မိမိ၏ ဖုန်း Gallery ထဲမှ ပုံများ ၊ အရေးကြီးသည့် ဖုန်းနံပါတ်များ ၊ အကောင့်များ အစရှိသည် တို့အား အားလုံး ရှင်းလင်းပြီးမှ ရောင်းချခြင်း ( သို့ ) တစ်ပါးသူထံအပ်နှံခြင်းအား ပြုလုပ်ပါ ။ မိမိ မမျှော်လင့် ထားသည့် အီးမေးလ်များ နှင့် ငွေကြေးတောင်းခံသည့် လင့်ခ်များအား မနှိပ်ပါနှင့်။ အကယ်၍ မိမိအကောင့်အား Hack ခံရခြင်း (သို့) Hack ခံရသည်ဟု သံသယရှိပါက မိမိ အကောင့်၏ စကားဝှက်အား အမြန်ပြောင်း၍ အခြားဝင်ရောက်အသုံးပြုနေသောဖုန်းများ / ကွန်ပျူတာများမှ Logout လုပ်ပါ

အထက်ပါ အချက်များသည် ဒစ်ဂျစ်တယ်လုံခြုံရေး အတွက် အခြေခံသိထားသင့်သည့် အလေ့အထ များဖြစ်ပြီး သင့်အနေဖြင့် အခြားသော အသေးစိတ် သိထားသင့်သည့် ဒစ်ဂျစ်တယ် လုံခြုံရေး သင်ခန်းစာများကိုလည်း ယခု Website တွင် ဆက်လက်လေ့လာနိုင်ပါသည်။
