Browser History ဖျက်ခြင်း

  • Post by
  • Feb 27, 2020

နေ့စဉ် မိမိတို့သည် browser မှတဆင့် Emailစစ်ခြင်း၊ Facebook အကောင့်ဝင်ခြင်း၊ Website ထဲသို့ဝင်ခြင်းများ ပြုလုပ်လျက်ရှိသည်။ ထို့သို့ပြုလုပ်ခြင်းသည် browser server များမှ မိမိ၏ အချက်အလက်များအား သိမ်းဆည်းရရှိထားနိုင်သည်။ အချိန်ကြာလာသည် နှင့်အမျှ ပြဿနာဖြစ်နိုင်စေပါသည်။ ထို့ကြောင့် မိမိ၏အကောင့်လုံခြုံရေး၊ မိမိ အချက်အလက်များအား သိမ်းဆည်းရရှိခံရမှုမှ ကင်းဝေစေရန် browser မှ Email, Facebook အကောင့်ဝင်ပြီးတိုင်း Logout ပြန်၍ ပြုလုပ်ခဲ့သင့်သည်။

မည်သည့်အတွက်ကြောင့် Browser History များရှင်းလင်းသင့်သည်ကို ဥပမာဖြင့် ဖော်ပြရလျှင် ဖူးပွင့်သည် သွေးအားနည်းတတ်သည့်အတွက် ဖူးပွင့် မေမေသည့် နေကြာစေ့စားခွင့်မပြုပါ ။ သို့သော် ဖူးပွင့်သည် နေကြာစေ့အရမ်းကြိုက်သည်။ နေကြာစေ့သည် သွေးအားနည်းခြင်းကိုလည်း မဖြစ်စေ ကြောင်း လေ့လာဖူးသည်။ နေကြာစေ့စားနေသည် ကို သူ့အမေတွေ့သောအခါ ဖူးပွင့်သည်အဆူခံရမည်မှာ ကျိန်းသေသည်။ ထို့ကြောင့် ဖူးပွင့်သည် နေကြာစေ့ထုတ်များ ၊ နေကြာစေ့ခွံများကို စားပြီးသည်နှင့် သေချာဂရုစိုက်ရှင်းလင်းရန်လိုအပ်ပါသည်။

ဤဖြစ်စဥ်တွင် ဖူးပွင့်သည် အသုံးပြုသူ ၊ နေကြာစေ့ခွံများသည် Browser History ဖြစ်သည်။

Safari

Safari ထဲသို့ဝင်ပါ။ Screen၏ ညာဘက်အောက်ဆုံးတွင်ရှိသော Book icon အားနှိပ်ပါ။ ထို့နောက် history page ၏အောက်ဆုံးညာဘက်ဒေါင့်တွင်ရှိသော clear မှတဆင့် မိမိပယ်ဖျက်လိုသော websiteအား browsingခဲ့သောအချိန်ကိုရွေးချယ်၍ အချက်အလက်များအား ပယ်ဖျက်နိုင်ပါသည်။ All time ကိုရွေးချယ်၍လည်း အချက်အလက် အားလုံးအား ပယ်ဖျက်နိုင်ပါသည်။ Safari home page ၏ အောက်ဆုံးညာဘက်ဒေါင့်ရှိ tabs iconမှတဆင့် ရွေးချယ်ပယ်ဖျက်နိုင်ပါပြီ။

Firefox

Firefox app ထဲသို့ဝင်ပါ။ Screen၏ အောက်ဆုံးညာဘက်ဒေါင့်တွင်ရှိသော menu icon ထဲသို့ဝင်၍ Your Libraryအားရွေးပါ။ Clear Recent History မှတဆင့် မိမိပယ်ဖျက်လိုသော website၏ browsing ခဲ့သောအချိနိအား ရွေးချယ်၍ ပယ်ဖျက်နိုင်ပါပြီ။ Firefox ၏ home pageတွင် screen၏ အောက်တွင်ရှိသော tabs iconအားနှိပ်ပါ။ ထို့နောက် screen၏ အောက်တွင်ရှိသော အမှိုက်ပုံးicon အားနှိပ်၍ မိမိ၏အချက်အလက်အားလုံးအားပယ်ဖျက်နိုင်ပါသည်။

Chrome

Chrome app ထဲသို့ဝင်ပါ။ Screen ၏ အောက်ဆုံးညာဘက်ထောင့် သို့မဟုတ် အပေါ် ညာဘက်ထောင့် တွင်ရှိသော အစက်သုံးစက် icon အားနှိပ်ပါ။ ထိုမှတဆင့် history ကိုနှိပ်ပါ။ ထို့နောက် Clear browsing data အားနှိပ်ပါ။ Time range မှတဆင့် မိမိပယ်ဖျက်လိုသော website၏ browsing ခဲ့သောအချိန်ကိုရွေး၍ ပယ်ဖျက်နိုင်ပါသည်။အချက်အလက်အားလုံးအားပယ်ဖျက်လိုပါက All time ကိုရွေးပါ။ Editမှတဆင့် မိမိပယ်ဖျက်လိုသော အချက်အလက်အားရွေးချယ်၍ ပယ်ချက်နိုင်ပါသည်။
